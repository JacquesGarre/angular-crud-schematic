# Angular schematic to create a crud component

## Requirements
    Git version 2.25+ (git --version)
    node v20.10+ (node --version)

# Installation
    git clone 
    npm install -g @angular/cli
    npm install -g @angular-devkit/schematics-cli
    npm install

# Usage

    cd crud
    schematics .:crud --dry-run=false
